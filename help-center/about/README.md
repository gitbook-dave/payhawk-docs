# About Payhawk

Payhawk is an all-in-one spend management platform that combines corporate cards, expense management, accounts payable, and integrations with your ERP or accounting software — in a single product.

## What Payhawk does

{% columns %}
{% column %}
### Spend control

Set budgets and policies before money is spent. Cards enforce limits automatically, so finance teams spend less time chasing receipts.
{% endcolumn %}

{% column %}
### Real-time visibility

Every transaction is captured the moment it happens, with receipts, VAT data, and accounting codes attached — no month-end scramble.
{% endcolumn %}
{% endcolumns %}

## Who uses Payhawk

| Role | What they do in Payhawk |
|------|------------------------|
| Administrator | Configures the account, manages users and teams, sets spend policies |
| Accountant | Reviews and exports transactions, manages reconciliation |
| Employee | Submits expenses, uses their Payhawk card, raises purchase requests |

## Key concepts

- **Fund account** — the balance your company tops up; cards draw from this pool
- **Spend policy** — rules that govern what can be spent, by whom, up to what limit
- **Expense** — any transaction that needs review, categorisation, and export to your accounting tool
- **Purchase order** — a pre-approved spend request before the money is committed
