# What's New

{% updates format="full" %}
{% update date="2025-01-15" tags="ai,feature" %}
## AI Office — Automated expense coding

AI Office now automatically suggests accounting codes, VAT rates, and cost centres based on merchant and category data. Accountants can accept suggestions in bulk or review individually.
{% endupdate %}

{% update date="2024-11-20" tags="feature" %}
## Trips and Travel AI Agent

Plan and book business travel entirely within Payhawk. The Travel AI Agent finds flights and hotels within policy, books on behalf of the traveller, and attaches itinerary data to the expense automatically.
{% endupdate %}

{% update date="2024-09-10" tags="api,feature" %}
## Purchase Orders API — v3

The Purchase Orders API now supports multi-line POs, custom approval workflows, and webhook events for every status transition. See the [API Reference](https://app.gitbook.com/s/6bSyETdqciHmrkb7hGG5/getting-started/authentication) for details.
{% endupdate %}

{% update date="2024-07-01" tags="feature" %}
## Payhawk in the US

Payhawk is now available for US-based companies. Issue USD cards, connect to US bank accounts, and sync with QuickBooks Online and NetSuite.
{% endupdate %}
{% endupdates %}
