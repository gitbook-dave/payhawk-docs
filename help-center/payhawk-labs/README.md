# Payhawk Labs

Payhawk Labs is where early-access and beta features live before they reach general availability. Labs features are opt-in — administrators can enable them from **Settings → Labs**.

{% hint style="info" %}
Labs features may change or be removed. They are not covered by the standard SLA.
{% endhint %}

## Currently in Labs

| Feature | Description | Status |
|---------|-------------|--------|
| AI expense coding | Automatic accounting code suggestions powered by AI | Beta |
| Travel AI Agent | Book flights and hotels within policy | Beta |
| Smart approvals | ML-based approval routing based on spend patterns | Early Access |

## Enabling Labs features

{% stepper %}
{% step %}
## Open Settings

Navigate to **Settings → Labs** in your Payhawk dashboard. Only administrators can access this page.
{% endstep %}

{% step %}
## Enable a feature

Toggle on the feature you want to try. Some features require additional configuration — follow the on-screen prompts.
{% endstep %}

{% step %}
## Share feedback

Use the in-app feedback button on any Labs feature to report issues or suggestions directly to the Payhawk product team.
{% endstep %}
{% endstepper %}
